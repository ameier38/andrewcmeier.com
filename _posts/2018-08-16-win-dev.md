## Windows Subsystem for Linux
Run bash on Windows.

### Installation
1) Open PowerShell.
2) Enable Windows Subsystem for Linux.
    ```
    > Enable-WindowsOptionalFeature -Online -FeatureName Microsoft-Windows-Subsystem-Linux
    ```
3) Install [Ubuntu](https://www.microsoft.com/en-us/p/ubuntu/9nblggh4msv6?rtc=1)

### References
- [Windows 10 Installation Guide ](https://docs.microsoft.com/en-us/windows/wsl/install-win10)

## Scoop
Command line installer for Windows.

### Installation
1) Open PowerShell.
2) Set the execution policy.
    ```
    > set-executionpolicy remotesigned -s currentuser
    ```
3) Install scoop.
    ```
    > iex (new-object net.webclient).downloadstring('https://get.scoop.sh')
    ```
4) Add extras and versions buckets.
    ```
    > scoop add bucket extras
    > scoop add bucket versions
    ```

### Usage
Search for packages
```
> scoop search vim
```
Install a package
```
> scoop install vim
```

### References
- [scoop GitHub](https://github.com/lukesampson/scoop)

## Cmder
Portable console emulator.

### Installation
1) Open PowerShell.
2) Install cmder.
    ```
    > scoop install cmder
    ```
3) Download [Meslo font](https://github.com/ryanoasis/nerd-fonts/blob/master/patched-fonts/Meslo/M/complete/Meslo%20LG%20M%20Regular%20Nerd%20Font%20Complete%20Windows%20Compatible.otf).
4) Right click on download font and click 'Install'.

    ![install_font](/assets/install_font.png)

5) Open cmder from start menu.

    ![open_cmder](/assets/open_cmder.png)

6) Open cmder settings using keyboard shortcut `Win-Alt-P`.
7) Set the font.

    ![set_cmder_font](/assets/set_cmder_font.png)

8) Set the PowerShell task.

    ![powershell_task](/assets/powershell_task.png)

9) Set the Ubuntu task. Change the path to use your username.

    ![ubuntu_task](/assets/ubuntu_task.png)

10) Pin cmder to the task bar. If it is the first item then you
can access it using the keyboard shortcut `Win-1`.

    ![pin_cmder](/assets/pin_cmder.png)

## posh-git, oh-my-posh
Better prompt.

### Installation
1) Open cmder.
2) Open PowerShell profile.
    ```
    > vim $PROFILE
    ```
3) Add posh-git and oh-my-posh.
    ```
    Import-Module posh-git
    Import-Module oh-my-posh
    Set-Theme paradox
    ```
4) Open a new shell and view the result.

    ![oh_my_posh](./images/oh_my_posh.png)

## Pipenv
Python package management.

### Installation
Install Pipenv. First make sure Python is installed (see [Python](#python) above).
```
> pip install --user pipenv
```

Check the installation.
```
> pipenv
Usage: pipenv [OPTIONS] COMMAND [ARGS]...

Options:
  --where             Output project home information.
...
```

### Usage
Create a virtual environment.
```console
> mkdir my-project
> cd my-project
> pipenv --python 3.7
```

Install a package
```console
> pipenv install pandas
```

### References
- [Pipenv homepage](https://docs.pipenv.org/)

## Jupyter Lab
Web based Notebook/REPL.
### Install
First create a directory, e.g. `notebooks`.
```console
> mkdir notebooks
> cd notebooks
```

Use Pipenv to create a virtual environment (see [Pipenv](#pipenv) above.).
```console
> pipenv --python 3.7
> pipenv install jupyterlab
```

### Usage
Start Jupyter
```console
> pipenv run jupyter lab
```

Install [Jupyter VIM extension](https://github.com/jwkvam/jupyterlab-vim) (I highly recommend).
```console
> jupyter labextension install jupyterlab_vim
```

## kubectl
Kubernetes CLI

### Install
1) Install `kubectl`
    ```
    > scoop install kubectl
    ```
2) Add your cluster configuration file to the `~/.kube` directory.
3) Set the `KUBECONFIG` environment variable.
    ```
    > vim $PROFILE
    ```
    ```powershell
    # "~\Documents\WindowsPowerShell\Microsoft.PowerShell_profile.ps1"
    $env:KUBECONFIG = "C:\Users\<username>\.kube\<my-config>;"
    ```

### Usage
```
> kubectl get pods 
```

## kubectxwin
CLI for managing Kubernetes contexts

### Install
1) Clone the repository.
    ```
    > git clone https://github.com/thomasliddledba/kubectxwin.git
    ```
2) Add the executable to the PATH.
    ```
    > vim $PROFILE
    ```
    ```powershell
    # "~\Documents\WindowsPowerShell\Microsoft.PowerShell_profile.ps1"
    $env:Path += ";C:\<path to kubectxwin repo>\bin"
    ```
3) Create an alias.
    ```
    > vim $PROFILE
    ```
    ```powershell
    # "~\Documents\WindowsPowerShell\Microsoft.PowerShell_profile.ps1"
    New-Alias ktx kubectxwin
    ```

### Usage
View contexts
```console
> ktx ls
```
```console
docker-for-desktop
other-context
```

Change context
```console
> ktx set docker-for-desktop
```
```console
Switched to context "docker-for-desktop".
```
